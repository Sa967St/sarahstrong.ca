Step 2 - Record the video

Navigation: Home Step 1 Step 3 Step 4

After you have all the software and hardware ready to go, here's how you would record the video.

  1. Open the drawing software.
  2. Set up any hardware you're using.
    • Plug in the USB cable from the drawing tablet.
    • Plug in the USB cable from the microphone.
    • Set up and start recording from the webcam.
  3. Start capturing the screen and audio.
  4. Deliver the lesson.
    • Use the pen and tablet to write and draw in the drawing software.
    • Speak into the microphone.
  5. Stop capturing the screen and audio.
  6. Turn off the webcam (if applicable).
  7. Save the files.
